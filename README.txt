Instagrambot накрутчик
Бот поможет накрутить подписчиков для вашего аккаунта.

Алгоритм работы: 
1)Вход в аккаунт(login и password, указать в authorization.py)
2)Переход на страницу с которым бот будет работать (с подписчиками этой страницы)
3)Сбор подписчиков с страницы который вы укажите в link.py
4)Фильтр собранных адресов 
5)Рандомное нажатие лайка на отфильтрованные ссылки пользователей
6)Подписка


Бот фильтрует пользователей по 3 критериям:
у пользователя аккаунт не должен быть закрытым,
не должно быть ссылки на странице,
минимум 10 публикаций на странице. 



Библиотека Silenium
webdriver: chromdriver.exe
